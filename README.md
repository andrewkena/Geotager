# Geotager

Десктопное приложение для записи GPS-координат в EXIF-метаданные фотографий.  
Работает на Windows, написано на Python + PyQt5.

![PyQt5](https://img.shields.io/badge/PyQt5-5.15-green)

## Возможности

- **Левая панель — фотографии**
  - Выбор папки с фотографиями (JPEG, PNG, TIFF, BMP, WebP)
  - Сетка миниатюр с изменяемым размером (слайдер)
  - Сортировка по имени, дате, размеру
  - Двойной клик — просмотр фото в полном размере с оверлеем: имя файла, дата съёмки, GPS-координаты и высота
  - Зелёный значок **GPS** на миниатюре, если у фото уже есть геоданные в EXIF
  - Правая кнопка мыши → **Исключить фото** (миниатюра серая, зачёркнута); повторно → **Вернуть**

- **Правая панель — геоданные**
  - Поддержка файлов: `.csv`, `.txt`, `.tsv`, `.dat`
  - Диалог настройки импорта:
    - Выбор разделителя (запятая, точка с запятой, табуляция, пробел, `|`, произвольный)
    - Пропуск первых N строк файла
    - Удаление полностью пустых столбцов
    - Предпросмотр с подсветкой дублирующихся строк красным
    - Маппинг столбцов: широта, долгота, высота
    - Выбор столбцов для отображения в таблице сопоставления
  - Сводка с количеством строк и числом дубликатов

- **Сопоставление и запись**
  - Кнопка «Сопоставить» — таблица, где каждому фото последовательно назначена строка с геоданными
  - Ширина столбцов таблицы регулируется вручную
  - Подтверждение перезаписи, если файлы в `geotag/` уже существуют
  - GPS-данные записываются в EXIF копий фотографий
  - Результат сохраняется в подпапку `geotag/` рядом с исходными фото
  - После завершения — кнопка **Открыть папку**

- **Интерфейс**
  - Три темы: Системная / Светлая / Тёмная (меню «Вид → Тема»)
  - Поддержка автоопределения тёмного режима Windows

## Установка

```bash
# Клонировать репозиторий
git clone https://github.com/andrewkena/Geotager.git
cd Geotager

# Создать виртуальное окружение и установить зависимости
python -m venv .venv
.venv\Scripts\pip install -r requirements.txt
```

## Запуск

```bash
.venv\Scripts\python main.py
```

## Формат файла геоданных

Любой текстовый файл с разделителями, содержащий столбцы с широтой и долготой.  
Пример CSV:

```
filename,latitude,longitude,altitude
IMG_001.jpg,55.751244,37.618423,156.0
IMG_002.jpg,55.752100,37.619000,158.5
```

Порядок строк в файле должен совпадать с порядком фотографий после сортировки.
