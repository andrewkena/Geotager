# Geotager

Десктопное приложение для записи GPS-координат в EXIF-метаданные фотографий.  
Работает на Windows, написано на Python + PyQt5.

## Возможности

### Левая панель — фотографии

- Выбор папки через меню **Файл → Открыть папку** или кнопку (`Ctrl+O`)
- Поддерживаемые форматы: JPEG, PNG, TIFF, BMP, WebP
- Сетка миниатюр с изменяемым размером (слайдер)
- Сортировка по имени, дате съёмки, размеру файла
- Зелёный значок **GPS** на миниатюре, если у фото уже есть геоданные в EXIF
- Двойной клик — полноэкранный просмотр с оверлеем: имя, дата, координаты, высота
- Правая кнопка мыши → **Исключить фото** (миниатюра серая, перечёркнута) / **Вернуть**
- Счётчик: `Фотографий: N (исключено: M)` — показывает активные и исключённые
- Кнопка **Переименовать…** — пакетное переименование

### Пакетное переименование

- Шаблоны с токенами: `{n}`, `{date}`, `{time}`, `{name}`, `{ext}`
- Готовые пресеты: «Имя + номер», «Дата + время + номер», «Только номер» и др.
- Настройка нумерации: начало, шаг, разрядность (001 / 01 / 1)
- Форматы даты: `YYYY-MM-DD`, `YYYYMMDD`, `DD.MM.YYYY`, `YYYY_MM_DD`
- Форматы времени: `HHmmss`, `HH-mm-ss`, `HH.mm.ss`
- Регистр расширения: как в оригинале / нижний / верхний
- Живой предпросмотр с подсветкой конфликтов имён
- Двухпроходное переименование (исключает коллизии A→B / B→A)

### Правая панель — геоданные

- Выбор файла через **Файл → Открыть файл геопривязки** или кнопку (`Ctrl+G`)
- Поддержка: `.csv`, `.txt`, `.tsv`, `.dat` и любых текстовых файлов с разделителями
- Диалог настройки импорта (открывается шириной основного окна):
  - Выбор разделителя: запятая, точка с запятой, табуляция, пробел, `|`, произвольный
  - Первая строка как заголовки / автоматические имена столбцов (A, B, C…)
  - Пропуск первых N строк
  - Удаление пустых столбцов
  - Предпросмотр с подсветкой дублирующихся строк
  - Маппинг столбцов: широта, долгота, высота (необязательна)
  - Выбор столбцов для отображения (чекбоксы): влияет на таблицу геоданных, карту и таблицу сопоставления
- Таблица показывает только отмеченные столбцы
- Сводка: количество строк и число дубликатов

### Карта геоточек

- Кнопка **Показать на карте** — открывает интерактивную карту
- Подложки: **Google Спутник** (по умолчанию), OpenStreetMap, Светлая, Тёмная, Esri Спутник
- Список точек в таблице справа от карты (столбцы из выбранных чекбоксов)
- **Наведение** на строку таблицы → маркер подсвечивается жёлтым на карте
- **Правый клик** на точке → **Исключить** (строка сереет, маркер тускнеет) / **Вернуть**
- При закрытии карты исключённые точки удаляются из таблицы геоданных

### Сопоставление и запись

- Кнопка **Сопоставить и записать геотеги →**
- Таблица сопоставления: каждому фото последовательно назначена строка геоданных
- Ширина столбцов настраивается вручную
- Подтверждение перезаписи, если файлы в `geotag/` уже существуют
- GPS записывается в EXIF копий фотографий (оригиналы не изменяются)
- Результат сохраняется в подпапку `geotag/` рядом с исходными фото
- По завершении — кнопка **Открыть папку**

### Интерфейс

- Меню: **Файл**, **Вид**, **Справка**
- **Файл**: открыть папку с фото (`Ctrl+O`), открыть файл геопривязки (`Ctrl+G`)
- **Вид**: три темы — Системная / Светлая / Тёмная; автоопределение тёмного режима Windows
- **Справка**: подробная инструкция (`F1`), о программе

## Установка

```bash
# Клонировать репозиторий
git clone https://github.com/andrewkena/Geotager.git
cd Geotager

# Создать виртуальное окружение и установить зависимости
python -m venv .venv
.venv\Scripts\pip install -r requirements.txt
```

## Запуск

```bash
.venv\Scripts\python main.py
```

## Формат файла геоданных

Любой текстовый файл с разделителями, содержащий столбцы с широтой и долготой.  
Порядок строк в файле соответствует порядку фотографий после сортировки.

Пример CSV с заголовками:

```
date,time,latitude,longitude,altitude
2025/11/28,08:45:08,55.005410787,73.326421307,467.39
2025/11/28,08:45:13,55.004806530,73.327803023,468.90
```

Пример без заголовков (столбцы автоматически называются A, B, C…):

```
2025/11/28 08:45:08 55.005410787 73.326421307 467.39
2025/11/28 08:45:13 55.004806530 73.327803023 468.90
```
